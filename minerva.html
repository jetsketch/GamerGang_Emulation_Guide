Minerva is basically just an archive of Myrient before the guy running it couldn’t sustain it anymore. (I HATE AI) Unlike Myrient, Minerva uses a torrent so they can bypass the problem Myrient had (having to spend a billion dollars on storage). It sounds a little daunting, but it’s really simple.

What You'll Need

qBittorrent

A VPN


(VPN is optional, but recommended since you are directly connecting to other people's PCs. Minerva is safe, but it's better safe than sorry. If you don’t have a vpn, dm me and I’ll send you my username and password)

For this example, we'll download Mario Kart Wii (USA)

  • Open this link to go to the r/roms megathread.
  • Find the console that the ROM is for. For Mario Kart Wii, it's the Nintendo Wii tab on the left side of the page. When entering, it should look like the photo on the left.
  • 99% of the time, you will just be using the first download link that is shown on the list. The rest are either redumps or used for some other niche purpose.
  • The page it sends you to should look like image on the left.
  • Where it says “search this directory” search up the game you want. In this case, Mario Kart Wii (USA)
  • Click on the correct game (pls make sure the region is correct).
  • The page that pops up should look like this, press “Torrent Download”.
  • Note: You don’t technically have to search up the game, you can press the first game you see and press “Torrent Download” but it helps to see what the file is named in the Torrent.
  • The page that pops up should look like the image on the left, press “Torrent Download” button.
  • Note: You don’t technically have to search up the game, you can press the first game you see and press “Torrent Download” but it helps to see what the file is named in the Torrent.

If you are using a VPN make sure you do this before opening the torrent file. If you arent, skip this part.

  • Open your VPN application, connect and enable port forwarding. Your port forwarding number will be displayed where I circled.
  • Open qBitorrent and go to Tools > Options > Connections and there will be an area to paste your port forwarding number to labeled “Port used for incoming connections:” Paste your port forwarding number there.
  • Press Apply and OK.
  • Now open the file you downloaded on the Minerva page (In this example, it’ll be called Minerva_Myrient - Redump - Nintendo - Wii - NKit RVZ [zstd-19-128k].torrent). It will automatically open qBitorrent.
  • To the right of the up pointing arrow by name, there should be a spot to resize the category. Resize it to the right so the name of the files show.
  • Press select none or unselect the Minerva_Myrient folder.
  • Search up the game you want (Mario Kart Wii (USA)) on the search bar
  • Click the box on the left of the game name.
  • Press OK and the game will automatically start downloading.
  • NOTE: It should default to your Downloads folder. If you don’t want that, change it where it says “save at”.
  • If you are downloading multiple games or a game with multiple disks, just make sure you press the check mark on them. (Ex: Final Fantasy VIII)
  • When your download is done, it will automatically start Seeding. All seeding does is share the downloaded files with other users people using the torrent. If you dont want to do this, just right click the torrent in qBittorrent and select "Stop". You can also click remove, just make sure "Also remove the content files" isn't selected or else it'll delete your rom.

That's basically it. After you're done downloading, the zip file for your game should have been saved in Minerva_Myrient > Redump > Nintendo - Wii - NKit RVZ [zstd-19-128k] (in this example). Just unzip it and plop that rom into whatever emulator you are using.
